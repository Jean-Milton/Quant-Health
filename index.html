One-time Setup
1. Go to strava.com/settings/api
2. Create an app — set callback URL to https://quant-health.netlify.app
3. Copy your Client ID and Client Secret
4. In Netlify dashboard → Site config → Environment variables:
   STRAVA_CLIENT_ID = your client ID
   STRAVA_CLIENT_SECRET = your client secret
5. Also add STRAVA_CLIENT_ID to this field:
